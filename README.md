Smart Juice Lab

A React Native mobile application that lets users order and customize fresh juice from an automated dispenser. The app communicates with an ESP32-based control system that manages ingredient reservoirs and dispensing mechanics in real time.

---

Overview

Smart Juice Lab bridges mobile software and embedded hardware into a single seamless experience. Users select flavors and quantities directly from their phone, and the app sends commands to an ESP32 microcontroller, which drives the physical dispensing process — no manual preparation required.

This project combines:
Mobile app development (React Native, Expo)
Embedded systems programming (ESP32)
Hardware-software communication (IoT)


Features

- Browse and select from multiple juice flavors
- Customize quantity per order
- Real-time communication with the ESP32 dispenser
- Cross-platform support (iOS & Android via Expo)
- Clean, responsive UI styled with Nativewind

---

Tech Stack

| Layer | Technology |
|---|---|
| Mobile App | React Native, Expo Go |
| Language | TypeScript |
| Styling | Nativewind (Tailwind CSS for React Native) |
| Hardware | ESP32 microcontroller |
| Communication | Wi-Fi|

---

How It Works

1. The user opens the app and selects a juice flavor and quantity.
2. The app sends the order as a command to the ESP32 over the network.
3. The ESP32 controls the ingredient reservoirs and dispensing mechanism to prepare the drink.
4. The app receives status updates (e.g. "dispensing," "complete") from the ESP32.
